# s4ngg portfolio

정적 HTML, Tailwind CSS CDN, JavaScript로 구성한 포트폴리오입니다.

## 로컬 실행

별도 빌드 과정 없이 `index.html`을 브라우저에서 열거나 정적 파일 서버로 실행할 수 있습니다.

## Vercel 배포

1. Vercel에서 이 GitHub 저장소를 Import합니다.
2. Framework Preset은 `Other`를 선택합니다.
3. Root Directory는 저장소 루트(`./`)로 둡니다.
4. Build Command와 Output Directory는 비워 둔 채 배포합니다.

`main` 브랜치에 푸시하면 연결된 Vercel 프로젝트가 자동으로 다시 배포됩니다.
